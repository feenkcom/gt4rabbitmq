# gt4rabbitmq
RabbitMQ HTTP API implementation

## Installation

```st
Metacello new
	repository: 'github://feenkcom/gt4rabbitmq:main/src';
	baseline: 'Gt4RabbitMq';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfGt4RabbitMq asClass loadLepiter
```
